cblaster.main.value_in_args(value, args)¶

Check for value in list of argument values, removing if found.

This function is used with arguments that take multiple values, mainly –binary.

It checks the list for a specific value (e.g. ‘hr’ or ‘he’). If found, the value is removed from the list via pop() and True is returned. If the value is not in the argument list, False is returned.

This facilitates users passing in values in different orders. For example, a user could specify:

–binary he path/to/filename hr

Since this function explicitly checks for and removes ‘he’ and ‘hr’, the validation function can still correctly parse the filename str.
